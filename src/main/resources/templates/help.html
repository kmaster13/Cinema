Авторизация/регистрация

Данное веб-приложение имеет два режима работы: режим администратора и клиента. При входе на сайт появляется окно авторизации, представленное на рисунке XX...

Если пользователь желает зарегистрироваться, ему необходимо нажать кнопку "Зарегистрироваться", чтобы открыть окно регистрации, представленное на рисунке ХХ...

Если пользователь забыл пароль, он может его восстановить, нажав на кнопку «Забыли пароль?». В открывшемся окне пользователь должен заполнить поле «Введите ваш email»...

Режим администратора

В состав режима администратора входят разделы: "Фильмы", "Создатели", "Места", "Заказы", которые представлены на рисунке ХХ...

Создание/Изменение фильма

Для создания фильма, администратору необходимо выбрать раздел «Фильмы»...

При нажатии кнопки «Добавить фильм» откроется страница для добавления нового фильма...

При нажатии кнопки «Подробнее» откроется страница «О фильме»...

Режим пользователя

После авторизации пользователь переходит на страницу Афиши, которая изображена на рисунке ХХ...

На этой странице можно выбрать сеанс, нажав на время, а также оставить отзыв...

После выбора мест пользователь попадает на страницу подтверждения брони, где показана вся информация по заказу...
